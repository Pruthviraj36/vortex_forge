# Changelog

## 1.0.3

- Renamed package to `vortex_forge`.
- Removed personal identifiers from package metadata and license.
- Updated global executable aliases to `vortex_forge` and `vx`.

## 1.0.1

- Publish readiness update: added package metadata and license.
- Added Nest-style command aliases: `generate`, `g`, and `module` alias for feature generation.
- Added executable alias `ff` for shorter command usage.

## 0.1.0

- Initial release.
- `create` command for GetX + MVC project shell.
- `make feature` command for module generation and route auto-wiring.
- Added stronger architecture scaffolding in generated apps: `core/base`, `core/config`, `core/network`, and `app/theme`.
- `make` now supports `screen`, `service`, `repository`, and `model` artifact generators.
- Feature generation now includes model, repository, and service by default.
